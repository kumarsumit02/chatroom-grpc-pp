# ts-node-grpc

A pet project made to understand the gRPC server setup. Just a simple chatroom application with online/offline status indication.

Tech Stack  - NodeJS, gRPC, React + TypeScript, Redis

## Project setup
```
yarn install
```

### Runs the gRPC server
```
yarn start
```

### Runs out client
```
yarn client
```
